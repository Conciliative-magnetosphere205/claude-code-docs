## 📚 What this is

Claude Code Docs is a document hub for Claude Code users. It brings useful guides into one place and keeps them in Korean and English.

Use it to find:

- Claude Code notes and guides
- Skill writing tips
- Workflow and structure ideas
- Analysis notes on source code
- English and Korean reference pages

This repository works as a reading hub. It helps you open the right document without searching across many sites.

## 🖥️ What you need

This repository is a document collection, so you do not install it like a normal app.

You only need:

- A Windows PC
- A web browser such as Chrome, Edge, or Firefox
- A stable internet connection
- A GitHub account if you want to star or fork the repo

If you want to keep a local copy, you can download the files from the GitHub page and open them with a text editor or Markdown viewer.

## 🪟 Windows setup steps

If you want a local copy on Windows, use these steps:

1. Open the GitHub page
2. Click Code
3. Click Download ZIP
4. Wait for the file to finish downloading
5. Open File Explorer

## 🛠️ Troubleshooting

### The ZIP file will not open

Try these steps:

1. Download the file again
2. Make sure the download finished
3. Use the built-in Windows unzip tool
4. Restart File Explorer

### The file opens in the wrong app

Right-click the file, choose Open with, then pick:

- Notepad
- VS Code
- Browser
- Another Markdown app

### The text looks hard to read

Try a Markdown viewer. It will show headings and links in a cleaner way than plain text editors.

### I only want one document

Open the GitHub page, then click the file you need. You do not need to download the whole repo if you only want one page.
